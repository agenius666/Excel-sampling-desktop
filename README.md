# SampleR

一个基于 **Tkinter + Pandas** 的桌面抽样处理工具，适用于 Excel / CSV 数据的规则筛选、抽样导出，以及结果去重处理。

---

## 功能概览

### 1. 抽样处理
- 支持读取 **Excel / CSV** 数据文件
- 支持处理 **单个文件** 或 **整个文件夹**
- 支持按规则文件执行：
  - 起止时间筛选
  - 条件筛选
  - 条件剔除
  - 指定字段前缀匹配
  - 字典替换 / 字典引用
  - 多种抽样方式
- 支持导出抽样结果到 Excel
- 自动生成处理日志，便于排查问题

### 2. 筛选去重
- 支持两个文件去重
- 支持两个文件夹内同名文件批量去重
- 输出去重后的 Excel 文件

### 3. 规则模板生成
- 内置规则模板生成功能
- 可直接在界面中生成 `.xlsx` 模板后填写规则

---

## 运行环境

建议环境：
- Python 3.10 及以上
- Windows 优先（Tkinter 桌面界面在 Windows 下使用最直接）

> `tkinter` 属于 Python 标准库，通常无需额外安装。  
> 若在 Linux 环境运行，可能需要先安装系统级 Tk 支持包。

```bash

```

- 选择抽样规则文件
- 选择抽样文件 / 文件夹
- 选择字典文件 / 文件夹（可选）
- 填写 Excel 工作表名称（可选）
- 执行抽样或筛选去重

## 使用说明

### 一、抽样处理

1. 打开程序
2. 选择 **抽样规则文件**
3. 选择 **抽样文件或文件夹**
4. 如有需要，选择 **字典文件或文件夹**
5. 如目标数据是 Excel 且需要指定工作表，可填写 **Excel 工作表名称**
6. 点击 **开始处理**
7. 选择结果保存位置，程序将输出抽样结果和日志文件

### 二、筛选去重

1. 点击 **筛选去重**
2. 选择：
   - 基准文件（A）/ 基准文件夹
   - 被筛选文件（B）/ 被筛选文件夹
3. 点击 **开始处理**
4. 保存去重结果

> 文件夹模式下，会按**同名文件**进行匹配和处理。

## 规则文件说明

程序会读取规则 Excel 中的关键列。当前版本代码中已识别并使用以下类型的列名：

- `#文件名`
- `#抽样方式`
- `#开始时间@字段名`
- `#结束时间@字段名`
- `#剔除@字段名`
- `#筛选@字段名`
- `#开头为@字段名`
- `#备注:...`

### 规则示例

程序自带模板生成功能，模板中已包含若干示例，例如：

- `#（@借方+@贷方）取绝对值后的前5`
- `#每个@科目名称（@借方+@贷方）取绝对值后每2月取前1`
- `#随机抽10个，随机数种子10`
- `#随机抽20个`

## 输入输出说明

### 输入
- 抽样规则文件：`.xlsx` / `.xls`
- 抽样数据文件：`.xlsx` / `.xls` / `.xlsm` / `.csv`
- 字典文件：文本文件（可选）

### 输出
- 抽样结果：`.xlsx`
- 去重结果：`.xlsx`
- 处理日志：`.log`

## 编码与兼容性

程序对 CSV 文件做了多编码尝试与自动检测，优先尝试：
- UTF-8
- UTF-8-SIG
- GBK
- GB2312
- 自动检测（`chardet`）

因此相对适合处理中英文混合来源的表格文件。
